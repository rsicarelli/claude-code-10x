Nova série acabou de sair do forno: Claude Code 101

"Eu nunca teria feito aquilo sozinho."

Essa frase me pegou de surpresa. Setembro de 2025, eu tinha acabado de mergear uma migração absurda: quase 10.000 testes adaptados, 2.000+ arquivos alterados, 50 mil linhas de código. Tudo em duas semanas. Num app mobile com milhões de usuários. Código de mais de 20 times.

Como? Quatro terminais rodando Claude Code em paralelo, cada um migrando uma fatia. 85% migrado de primeira.

Mas o que realmente me marcou não foi a velocidade. Foi perceber que meu papel tinha mudado. Eu não estava mais na linha de produção apertando parafuso. Eu estava projetando a fábrica.

Escrevi essa série pra desmontar o que está por trás disso. De onde veio a programação agêntica, como funciona, e o que você precisa aprender pra usar de verdade. Sem hype, sem medo, com dados honestos (inclusive os que mostram que nem tudo são flores).

A série Claude Code 101 completa:

1. Introdução à Programação Agêntica (este!)
2. Desmistificando os Modelos de Linguagem
3. Por dentro do Claude Code
4. Context window: o recurso mais precioso do agente
5. Seu primeiro projeto com Claude Code
6. A arte de dar instruções pro agente
7. Ensinando o agente a trabalhar do seu jeito
8. Harness: como o agente funciona e o que você configura
